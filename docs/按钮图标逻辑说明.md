# 按钮图标显示逻辑

## iOS 项目逻辑（MainContainerViewController.swift）

### 左侧按钮（modeSwitchButton）
| 当前界面 | 显示图标 | 点击后跳转 |
|---------|---------|-----------|
| 我的安排 (calendar) | 🌿 植物图标 | → 植物界面 |
| 植物 (plant) | 📅 日历图标 | → 我的安排 |
| 我的 (profile) | 📅 日历图标 | → 我的安排 |

### 右侧按钮（profileButton）
| 当前界面 | 显示图标 | 点击后跳转 |
|---------|---------|-----------|
| 我的安排 (calendar) | 👤 个人图标 | → 我的 |
| 植物 (plant) | 👤 个人图标 | → 我的 |
| 我的 (profile) | 🌿 植物图标 | → 植物界面 |
