# 气泡球窗口 · 调参说明

改**整体大小**时必须**两处一致**（逻辑像素）：

1. **`src-tauri/tauri.conf.json`** → `app.windows` 里 **`label` 为 `"bubble"`** 的 **`width`** / **`height`**（须相同，否则不是正圆裁切）。
2. 前端 **`src/BubbleBall.tsx`** 使用 `h-screen w-screen` 铺满该窗口，一般**不必**再写死像素。

可选调整：

| 想改的效果 | 改哪里 |
|------------|--------|
| 更小 / 更大 | 只改 `tauri.conf.json` 里 bubble 的 `width`/`height`（例如 56～96） |
| 系统阴影 | `shadow: false` 与 `src-tauri/src/lib.rs` 里 `bubble.set_shadow(false)` |
| 图在圆内裁切方式 | `BubbleBall.tsx` 里 `<img>` 的 `object-contain`（完整显示）↔ `object-cover`（铺满圆，可能裁边） |
| 右键功能菜单 | `BubbleBall.tsx`：`BubbleContextMenu`（Portal 挂 `body`）、`MENU_ESTIMATE_*`（贴边）、`clampMenuPosition` |
| 背景图资源 | `images/光合日历气泡球背景.png` |

改完保存后重新 **`npm run tauri:dev`**（或让 Tauri 热重载）即可。
